# Procédure d'installation de Java sur Windows, MacOS et Linux
## Installation de java sur Windows
1. Accédez au site officiel de [Java Oracle](https://www.oracle.com/java/technologies/downloads/#jdk20-windows)
2. Téléchargez le **JDK** (Java Development Kit) correspondant à votre système d'exploitation et à l'architecture de votre machine (32 bits ou 64 bits).
3. Double-cliquez sur le fichier téléchargé et suivez les instructions de l'assistant d'installation.
4. Une fois l'installation terminée, vérifiez si Java est correctement installé en ouvrant une fenêtre de commande (Invite de commandes) et en exécutant la commande suivante :
```sh
java -version
```
## Installation de java sur Windows
1. Accédez au site officiel de [Java Oracle](https://www.oracle.com/java/technologies/downloads/#jdk20-windows)
2. Téléchargez le **JDK** (Java Development Kit) correspondant à votre version de macOS.
3. Double-cliquez sur le fichier téléchargé(.dmg) et suivez les instructions de l'assistant d'installation.
4. Une fois l'installation terminée, vérifiez si Java est correctement installé en ouvrant une fenêtre de commande (Invite de commandes) et en exécutant la commande suivante :
```sh
java -version
```
## Installation de java sur Linux
1. Ouvrez un terminal
2. Exécutez les commandes suivantes pour mettre à jour les paquets disponibles :
```sh
sudo apt update
```
3. Exécutez la commande suivante pour installer JDK
```sh
sudo apt install default-jdk
```
4. Une fois l'installation terminée, vérifiez si Java est correctement installé en ouvrant une fenêtre de commande (Invite de commandes) et en exécutant la commande suivante :
```sh
java -version
```
